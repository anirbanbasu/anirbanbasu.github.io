[Frankfurter](https://frankfurter.dev/) is a useful API for latest currency exchange rates, historical data, or time series published by sources such as the European Central Bank (ECB).

Around mid-2025, I made the Frankfurter MCP project in Python utilising [FastMCP](https://gofastmcp.com/). It exposes tools for fiat currency rates and conversions over the Model Context Protocol (MCP). It allows you to easily integrate currency exchange functionality into your LLM-based applications.

You can find the following links related to the Frankfurter MCP project.

 - <a href="https://github.com/anirbanbasu/frankfurtermcp" target="_blank">GitHub repository</a> for the Frankfurter MCP project.
 - <a href="https://pypi.org/project/frankfurtermcp/" target="_blank">PyPI package</a> for the Frankfurter MCP project.
 - <a href="https://frankfurtermcp.fastmcp.app/mcp" target="_blank">Cloud-based</a> Frankfurter MCP, hosted by the FastMCP cloud.
